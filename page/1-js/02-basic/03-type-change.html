형변환

문자형으로 변환: let str = String(value)

alert는 자동으로 문자형으로 변환함

숫자형으로 변환: let num = Number(value)

숫자로된 문자("212")는 /, *, - 를 만나면 숫자로 변환됨

전달받은 값 형 변환 후
undefined NaN
null 0
true / false 1 / 0
string 전달받은 문자열을 “그대로” 읽되, 처음과 끝의 공백을 무시합니다. 문자열이 비어있다면 0이 되고, 오류 발생 시 NaN이 됩니다. 
Number("   123   "); // 123
Number(true); // 1
Number(false); // 0
Number("123z"); // NaN

Boolean : Boolean(value)

0, "", null, undefined, NaN : false

그외, " ", "false", "null" 등은 : true
